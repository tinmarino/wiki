# Dark background

* Font : 2px, 3px larger

# Typographic

* Kerning : reduce space between some letter like `Vi`
* Leading : change line space
* Tracking : change distance between __all__ letters (use with caution)
  * See `letter-spacing` and `word-spacing` properties
